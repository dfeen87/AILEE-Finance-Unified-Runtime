# Layer 13 — Deterministic Stress‑Regime Override

## Architecture Specification

Layer 13 acts as AILEE’s **crash‑mode governor**: when the system enters a defined stress regime, Layer 13 applies **hard overrides** on top of normal behavior, with no ambiguity and no improvisation.

Sits above all other governance layers (Layers 8–12), ensuring deterministic de‑risking, exposure freeze thresholds, and fallback compression to defensive baselines when under extreme market distress.

The system operates under the version tag `STRESS_REGIME_OVERRIDE_V1`.

---

### 1. Deterministic Stress‑Mode Entry/Exit Pipeline

The stress‑regime override evaluates a composite set of conditions from prior layers:
- `volatility_index` (Layer 9/stabilizer)
- `drawdown_index` (Layer 9/stabilizer)
- `correlation_index` (Layer 11)
- `temporal_residual_sum` (Layer 12)

A stress mode or crisis mode becomes active if **any** of these key metrics exceed their configured thresholds:
- `volatility_index > volatility_threshold`
- OR `drawdown_index > drawdown_threshold`
- OR `correlation_index > correlation_threshold`
- OR `temporal_residual_sum > residual_threshold`

#### Stress Level Hierarchy
- `NORMAL (0)`: No thresholds breached.
- `STRESS (1)`: Lower-bound stress detected.
- `CRISIS (2)`: One or more key thresholds exceeded or forced from high-level governor state.

---

### 2. Deterministic Crash Dampening

When entering or inside `STRESS` or `CRISIS` mode, apply **strong, deterministic scaling** to all risk‑bearing allocations:
$$ w_{i, t+1} \leftarrow w_{i, t+1} \times s_{\text{crash}} $$

where $s_{\text{crash}} \in (0, 1)$ is a fixed, versioned factor (e.g., `0.5`).

- An asset is defined as risk-bearing if:
  - It is not CASH (`AssetId::CASH`)
  - Its asset-level `risk_score > 40.0`

---

### 3. Deterministic Exposure Freeze Thresholds

To prevent increasing risk during severe volatility or drawdown:
- If `volatility_index > 0.5 * volatility_threshold` OR `drawdown_index > 0.5 * drawdown_threshold` (representing entry into the STRESS regime), **exposure freeze is engaged**.
- Once engaged, for any risk-bearing asset, the proposed allocation $w_{i, t+1}$ cannot exceed its previous allocation $w_{i, t}$:
  $$ w_{i, t+1} \leftarrow \min(w_{i, t+1}, w_{i, t}) $$
- De‑risking (reducing allocation) is always allowed.

---

### 4. Deterministic Fallback Compression

If normal routing/constraints cannot bring the portfolio back inside safe bounds under stress (or if `normal_safety_failed == true` or `effective_mode == CRISIS`), Layer 13 applies a **fallback compression** toward a versioned `SafeBaseline` allocation:
$$ w_{i, t+1} \leftarrow (1 - \lambda) \cdot w_{i, t+1} + \lambda \cdot w^{\text{safe}}_i $$

where $\lambda \in [0, 1]$ is a fixed, versioned factor (e.g., `0.7`) and $w^{\text{safe}}_i$ is the baseline allocation for that asset (e.g., 80% CASH, 20% GOLD).

---

### 5. Struct Definitions (64-byte Cache-Aligned)

All Layer 13 data structures are designed to be exactly 64 bytes with explicit padding and alignment:

```cpp
enum class StressMode : uint8_t {
    NORMAL = 0,
    STRESS = 1,
    CRISIS = 2
};

struct alignas(64) StressOverrideRules {
    double volatility_threshold;         // 8 bytes
    double drawdown_threshold;           // 8 bytes
    double correlation_threshold;        // 8 bytes
    double residual_threshold;           // 8 bytes
    double crash_dampening_factor;       // 8 bytes (s_crash)
    double fallback_lambda;              // 8 bytes (lambda)
    uint8_t mode;                        // 1 byte (NORMAL/STRESS/CRISIS)
    uint8_t reserved[15];                // 15 bytes padding -> 64 bytes total
};
static_assert(sizeof(StressOverrideRules) == 64, "StressOverrideRules must be exactly 64 bytes");

struct alignas(64) SafeBaseline {
    AssetId asset_id;                    // 2 bytes
    uint8_t flags;                       // 1 byte (bit 0: IS_SAFE_SET)
    uint8_t _padding1[5];                // 5 bytes alignment
    double baseline_allocation;          // 8 bytes (e.g., CASH=0.8, others=0.0)
    uint8_t reserved[48];                // 48 bytes padding -> 64 bytes total
};
static_assert(sizeof(SafeBaseline) == 64, "SafeBaseline must be exactly 64 bytes");

struct alignas(64) StressTraceStep {
    AssetId asset_id;                    // 2 bytes
    uint8_t flags;                       // 1 byte (bit 0: dampened, bit 1: frozen, bit 2: fallback_applied)
    uint8_t _padding1[5];                // 5 bytes alignment
    double original_allocation;          // 8 bytes (before override)
    double adjusted_allocation;          // 8 bytes (after override)
    uint8_t reserved[40];                // 40 bytes padding -> 64 bytes total
};
static_assert(sizeof(StressTraceStep) == 64, "StressTraceStep must be exactly 64 bytes");

struct alignas(64) StressPortfolioState {
    double portfolio_risk;               // 8 bytes
    double stress_index;                 // 8 bytes (composite stress metric)
    double volatility_index;             // 8 bytes
    double drawdown_index;               // 8 bytes
    double correlation_index;            // 8 bytes
    double temporal_residual_sum;        // 8 bytes
    uint8_t stress_level;                // 1 byte (NORMAL/STRESS/CRISIS)
    uint8_t reserved[15];                // 15 bytes padding -> 64 bytes total
};
static_assert(sizeof(StressPortfolioState) == 64, "StressPortfolioState must be exactly 64 bytes");
```
